# Projeto-Elder-geek
Elder geek é um projeto de site criado no curso de informatica do colégio ULBRA são lucas, com a proposta de ser uma rede social para pessoas com mais de 60 anos e que são nerds/geeks.
# Visão geral
 esta aplicação está sendo desenvolvida para a disciplina de análise de sistema, para a o curso técnico em informatica. A prática será do planejameto do projeto em questão, serão desenvolvidos todo o planejamento, diagramas da UML( caso uso; Diagrama sequência, Diagrama de classes), Diagrama entidade relacionamento (DER). Protótipo das telas( Uso de wireframes / mockups ); Requisitos não funcionais, requisitos funcionais e restrições .

permite ao usuário criar uma conta, inserindo seu nome, sobrenome, data de nascimento, criando uma senha, colocando seu email e gênero
o usuário consegue logar clicando em "Já possui uma conta?"

Funcionalidades:

Login e autenticação de usuário
Exibição de informações do usuário
Envio de solicitações de amizade
Bloqueio de outros usuários
Notificações para o usuário
Acesso a outras páginas do sistema
Qualidades:

Fácil de usar e intuitiva
Acessível para pessoas com deficiências
Responsiva a diferentes tamanhos de tela
Segura e protegida
Performante e com carregamento rápido
Esteticamente agradável e com bom design
Observações:

Atender a todos os requisitos garante uma interface útil e eficaz para os usuários.
Os requisitos funcionais definem o que a interface deve fazer, enquanto os não funcionais definem como ela deve fazer isso.
